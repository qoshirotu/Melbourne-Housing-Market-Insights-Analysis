# Melbourne Housing Market Analysis

A Python-based exploratory data analysis project that examines property prices, housing characteristics, suburb trends, seller activity, and property recommendations using the Melbourne Housing dataset.

## Overview

This project analyzes housing listings in Melbourne to identify patterns in property prices and market characteristics. The analysis focuses on variables such as property price, number of rooms, land size, distance from the CBD, suburb, property type, and seller.

The project applies exploratory data analysis techniques to detect outliers, examine correlations, compare suburbs, evaluate seller activity, and generate rule-based property recommendations.

> Note: This project focuses on data analysis and recommendation logic. It does not build a predictive machine learning model.

## Objectives

- Explore the Melbourne housing dataset.
- Identify missing values and inspect data types.
- Analyze property price distributions.
- Detect price outliers using IQR and Z-score methods.
- Examine correlations between numerical property features.
- Compare housing prices across suburbs.
- Analyze price relationships with rooms and distance from the CBD.
- Identify sellers with the highest number of transactions.
- Generate property recommendations based on buyer criteria.
- Export analysis results into an Excel report.

## Dataset

This project uses the **Melbourne Housing Full Dataset**.
